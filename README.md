# Super Shahad Portfolio  
A Mario-themed personal portfolio website built for Assignment-4.  
This project showcases my skills, projects, GitHub API integration, animations, sound effects, and interactive features — all wrapped in a fun Super Mario design.


## Features

### Mario-Themed UI  
- Retro pixel style  
- Animated question blocks  
- Hover & click animations  
- Custom Mario fonts  
- Coin sound effect when skills unlock  

### Dynamic Greeting  
- Stores your name using `localStorage`  
- Auto-greets the user based on time of day  
- “Welcome Back” section that remembers visits  

### Skills Block System  
- Clickable Mario blocks  
- Reveals skills with animation  
- Saves progress so skills stay unlocked  
- Includes HTML, CSS, JavaScript, Responsive Design, Python, MySQL, Java, GitHub, etc.

### Projects Section  
- Fully dynamic  
- Live search  
- Level filters (Beginner / Intermediate / Advanced)  
- Sort by title or date  
- Auto-renders project cards with images  

### GitHub API Integration  
- Shows my latest repositories  
- Loads directly from GitHub  
- Each repo includes a description + link  

### Contact Form  
- Validation for name, email, and message  
- Error messages  
- Success feedback  

### Theme Toggle  
- Light / dark mode using `data-theme`  
- Saves preference  

### User Tracking  
- Visit counter  
- Timer showing how long you've been on the page
